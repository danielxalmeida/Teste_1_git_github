Daniel Almeida



1 - O que é o HTML e para que serve?

H --» Hyper
T --» Text
M --» Markup
L --» Language

O HTML é uma linguagem de marcação que descreve a estrutura de uma página web e a principal linguagem de criação de páginas web.
É um conjunto de instruções (tags, elementos, referências) que indicam ao browser como apresentar e estruturar os conteúdos.
O conteúdo é composto por elementos (ou tags). Variam desde titulos, paragrafos, imagens, links, tabelas, botões...
Cada elemento pode ser personalizado individualmente, sob a forma de atributos, de forma a ser apresentado como pretendido (cor, letra, fundo, tamanho, formatação...).

2 - Quais ferramentas são necessárias para construir um site em HTML?

Para construír um site HTML, basta um simples editor de texto, tipo Wordpad ou Notepad.
Para visualizar o resultado, é ainda necessario um Browser.
No entanto, para facilitar a escrita e a edição, podem ser utilizados outros softwares, especialmente para Web ou para código.
De momento, o VScode é uma ajuda, pois lista alguns elementos e atributos aquando se inicia a sua digitação, ou preenche automaticamente o início e o fim de cada elemento.

3 - Quais os elementos base necessários para a criação de uma página web em HTML?

São 4 os elementos base necessários para a criação de uma pagina HTML:

  1. <Doctype html>
    2. Define o tipo de documento e indica ao browser o tipo de página e a linguagem utilizada.
  2. <html></>
    3. Definição da linguagem e delimita todo o conteúdo, visível ou não, de uma página. Dentro do HTML podemos ter mais dois elementos base: <head></> e <body></>
  3. <head></>
    4. O campo "head" agrupa, essencialmente, os metadados necessários para a página e para os navegadores e motores de busca. Podem ser inseridos títulos (nome da página), tipo de alfabeto (utf-8), palavras-chave para pesquisa, descrições do site, do conteúdo... Os metadados aqui inseridos, não serão visíveis no corpo da página.
  4. <body></>
    5. Conteúdo visível da página. É composto por vários elementos/Tags e respectivos atributos, de forma a formatar e personalizar toda a informação a apresentar. Pode variar entre texto, imagens, links e muito outro conteúdo.

4 - O que é um elemento? Dê exemplos.

Os elementos são a base da construção do HTML. Cada elemento representa uma parte do conteúdo.
São responsáveis por todas as partes (Visíveis e invisíveis) de um página HTML.
Um elemento composto é composto por 3 partes:

  1. Tag de abertura
    2. Os elementos iniciam-se com "< _Instrução_ >".
    A instrução define o elemento.
    Após a instrução, podem ainda ser adicionados atributos:
    "< _Instrução_ _Atributos_>"
    Os atributos (um ou mais) são adicionados depois da instrução e podem variar de cor, tamanho, posição...
    Em suma, definem a apresentação/formatação do elemento.

  2. Conteúdo
    3. Esta á a parte visivel do elemento e a parte a ser formatada pelos atributos.

  3. Tag de fechamento
    4. Delimita o fim do elemento, bem como da formatação e atributos definidos.
    É representada apenas por "</ _Instrução_ >"

Existem, também, tags sem fechamento, que vamos explorar na questão seguinte.

Exemplos de tags compostas:

<p>Tag de parágrafo</p> 

A tag <p></p> é um exemplo do que é e de como funcionam os elementos. 
Tag de abertura: <p>
Conteúdo: Todo o conteúdo que queremos apresentar;
Tag de fecho: </p>

Em HTML:
<p>
    Conteúdo, incluindo imagens, links, videos...
</p>

5 - O elemento < br > necessita ser fechado? Justifique.

<br>, tal como <hr> ou <img>, são elementos vazios, que não têm fecho.
São chamadas de elementos vazios, pois não contêm nenhum conteúdo.
No entanto, há elementos vazios que podem conter atributos, como por exemplo a tag <img>.

Os elementos vazios têm uma função específica pré definida, e como tal, não permitem conteúdo nem precisam de ser fechados, pois a sua função é única e aplicada no momento.
Por exemplo, o elemento <br> introduz uma quebra de página.
Não permite conteúdo e, como tal, não necessita de ser fechado (por ser um elemento vazio, por ter uma função pre-definida e por não permitir conteúdo).

O elemento <img>, sendo um elemento vazio, vai necessitar de ter atributos, tais como:

--» caminho para a imagem src="xxxx.xxx";
--» texto alternativo alt="xxx";
--» definição do tamanho a apresentar (width/height).

6 - Para que servem os atributos? Dê exemplos.

Os atributos servem para fornecer informação adicional sobre o elemento.
São eles que determinam a forma como o browser vai exibir os elementos.
Podem ser de diferentes  funções, tais como:

        entre outros
    
    
      referência --» href ou scr
        
        Definem o caminho para o link ou imagem.
    
        
      ou de informação
